# 配置根域名 bio-chain.cn 的 DNS 记录

## 如果需要在 GitHub Pages 中使用根域名

在域名管理界面的 "添加子域名" 部分，添加 4 条 A 记录：

### 记录 1
- **子域名**：留空或 `@`（表示根域名）
- **解析类型**：`A/AAAA`
- **DNS 指向**：`185.199.108.153`
- **TTL/Prior**：`7200`
- 点击 "添加子域名"

### 记录 2
- **子域名**：留空或 `@`
- **解析类型**：`A/AAAA`
- **DNS 指向**：`185.199.109.153`
- **TTL/Prior**：`7200`
- 点击 "添加子域名"

### 记录 3
- **子域名**：留空或 `@`
- **解析类型**：`A/AAAA`
- **DNS 指向**：`185.199.110.153`
- **TTL/Prior**：`7200`
- 点击 "添加子域名"

### 记录 4
- **子域名**：留空或 `@`
- **解析类型**：`A/AAAA`
- **DNS 指向**：`185.199.111.153`
- **TTL/Prior**：`7200`
- 点击 "添加子域名"

## 重要提示

⚠️ **需要先删除现有的根域名 A 记录**：
- 删除 `BIO-CHAIN.CN` → `99.83.190.102`
- 删除 `BIO-CHAIN.CN` → `75.2.60.5`

然后再添加上面的 4 条新记录。

## 更新 CNAME 文件

如果同时使用根域名和 www，CNAME 文件应该包含：
```
www.bio-chain.cn
bio-chain.cn
```

## 在 GitHub Pages 中配置

1. Custom domain 输入：`bio-chain.cn`（不带 www）
2. 等待 DNS 验证（可能需要 30 分钟到 2 小时）
3. 点击 "Check again" 刷新验证
