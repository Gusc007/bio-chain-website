## 📊 功能特性

### 邮件发送流程
1. **客户提交表单** → 前端验证
2. **发送到服务器** → `/api/contact` 接口
3. **服务器验证** → 数据格式和内容验证
4. **发送主邮件** → cindy.zhang@bio-chain.cn
5. **发送确认邮件** → 客户邮箱
6. **返回响应** → 成功/失败状态

### 邮件内容
- **主邮件**: 包含客户完整信息（姓名、邮箱、电话、服务类型、详细需求）
- **确认邮件**: 感谢客户咨询，包含联系方式和回复时间
- **格式**: HTML 格式，专业美观

### 安全特性
- **数据验证**: 邮箱格式、电话格式验证
- **错误处理**: 完善的错误处理和回退机制
- **日志记录**: 详细的发送日志和错误日志
- **环境变量**: 敏感信息通过环境变量管理
