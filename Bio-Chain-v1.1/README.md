## 🎨 自定义指南

### 修改颜色主题
在 `css/style.css` 中修改颜色值：

```css
.logo {
    color: #2563eb; /* 主色调 */
}

.logo i {
    color: #f97316; /* 强调色 */
}
```

### 添加服务
在 `index.html` 的服务区域添加新的 `.service`：

```html
<div class="service">
    <i class="fas fa-icon-name"></i>
    <h3>服务名称</h3>
    <p>服务描述</p>
</div>
```

### 添加专业领域
在 `index.html` 的专业领域区域添加新的 `.expertise-item`：

```html
<div class="expertise-item">
    <i class="fas fa-icon-name"></i>
    <h3>专业领域名称</h3>
    <p>详细描述</p>
</div>
```
